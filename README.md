# Web Meta Analyzer Pro 🔍

[![Deploy with Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/yourusername/web-meta-analyzer)

**Web Meta Analyzer Pro** is an advanced tool for analyzing website meta information including Title, Canonical URL, AMP detection, and HTTP Status with CORS proxy support and user-agent rotation.

## ✨ Features

### Core Features
- ✅ **Single URL Analysis** - Check one URL at a time
- ✅ **Batch URL Analysis** - Check multiple URLs simultaneously
- ✅ **Title Extraction** - Extract title tag accurately
- ✅ **Canonical Detection** - Detect `<link rel="canonical">`
- ✅ **AMP Detection** - Detect AMP (amphtml link, AMP attribute, boilerplate)
- ✅ **HTTP Status** - Show status code, status text, and response time
- ✅ **CORS Proxy** - Multiple proxy fallback for CORS issues
- ✅ **User-Agent Rotation** - Rotate user-agents to avoid blocking

### Advanced Features
- 🔄 **Retry Mechanism** - Automatic retry on failure (2 attempts)
- 🌐 **Multiple CORS Proxies** - Fallback if direct fetch fails
- 📊 **Progress Bar** - Visual progress indicator
- ⏱️ **Rate Limiting** - 500ms delay between requests
- 📥 **Export Options** - CSV and JSON export
- 📈 **Summary Statistics** - Comprehensive analysis summary
- 🎯 **Response Time Tracking** - Measure page load speed
